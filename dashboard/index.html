Raw Sensor Data

Direct readings from the three primary sensors

AS7262 - 6-Channel Spectral Sensor

Sensor Description: The AS7262 is a 6-channel visible light spectral sensor that measures light intensity at specific wavelengths (450nm, 500nm, 550nm, 570nm, 600nm, 650nm).

Data Usage: Spectral data is used to calculate color temperature and tint by analyzing the distribution of light across different wavelengths.

TSL2591 - High-Precision Luminance Sensor

Sensor Description: The TSL2591 is a very high sensitivity light-to-digital converter that measures luminance (lux), infrared, and full spectrum light.

Data Usage: Luminance data helps calibrate color temperature calculations in different lighting conditions and provides exposure information.

DFRobot SEN0611 - CCT & ALS Sensor

Sensor Description: The SEN0611 measures Correlated Color Temperature (CCT) in Kelvin and Ambient Light Sensing (ALS) in lux.

Data Usage: CCT data provides a direct color temperature reading that helps validate and calibrate the calculations from spectral data.

GPS Module (Optional)

Module Description: The GPS module provides location data including latitude, longitude, altitude, number of satellites, and HDOP.

Data Usage: Location data can be used to correlate color measurements with geographical position and time of day.
